# KittyTunnel
A terminal application that uses pytubefix to create a convenient youtube downloader with some neat extras.

the name was really just inspired by youtube, replacing you with a cat, and tube with a tunnel.

Recommendations:
- I recommend using a vpn while using my tool, as this will generally allow for better online safety.
- I also suggest that when sharing links, you put them into my programm and share the shortened link, as it has youtubes link origin tracking removed.
- If your link wont work, try changing to a different device, location or IP.
-alternatively you can also attempt a different video link with the same content.
